# Google Chat App

Google Chat doesn't have official application for MacOS.

## Installation

Use the package manager [pnpm](https://pnpm.io/) to install this app.

```bash
pnpm i
```

## Usage
Build with:
```bash
pnpm build
```
In `/dist` folder will be generated .dmg file. Install as any other MacOS app.
